# Zasada działania czujnika APS11450

## Cel dokumentu

Dokument opisuje działanie APS11450 z punktu widzenia implementacji oprogramowania i diagnostyki.

## 1. Ogólna zasada działania

APS11450 jest trójprzewodowym planarnym czujnikiem Halla zgodnym z ISO 26262 (SEooC, ASIL B). Czujnik wykrywa obecność pola magnetycznego i udostępnia stan wyjścia oraz informacje diagnostyczne poprzez poziom napięcia wyjściowego.

## 2. Cykl pracy

1. Autodiagnostyka.
2. Pomiar pola magnetycznego.
3. Porównanie z progami BOP i BRP.
4. Aktualizacja wyjścia.
5. Diagnostyka wewnętrzna.
6. Przejście do FAULT w razie błędu.
7. Powtarzanie cyklu.

## 3. Logika magnetyczna

- BOP -- próg przełączenia.
- BRP -- próg powrotu.
- BHYS = BOP - BRP.

Dostępne warianty: Unipolar North, Unipolar South, Omnipolar, Latching.

## 4. Co widzi mikrokontroler

Nie widzi:

- wartości pola,
- odległości od magnesu,
- kierunku ruchu,
- prędkości ruchu.

Widzi:

- VOUT(LOW),
- VOUT(HIGH),
- VOUT(FAULT),
- niepoprawne poziomy napięcia (przy użyciu ADC).

## 5. Poprawne stany wyjścia

| Stan | Typowe napięcie |
| --- | --- |
| VOUT(LOW) | ok. 20% VPU |
| VOUT(HIGH) | ok. 80% VPU |
| VOUT(FAULT) | VPU |

Znaczenie LOW/HIGH zależy od wariantu czujnika i aplikacji.

## 6. Obszary napięć

| Zakres | Znaczenie |
| --- | --- |
| 0-10% VPU | Potencjalna usterka zewnętrzna |
| 10-30% VPU | Poprawny LOW |
| 30-70% VPU | Potencjalna usterka zewnętrzna |
| 70-90% VPU | Poprawny HIGH |
| 90-100% VPU | HIGH lub FAULT zgodnie z dokumentacją |

Każde napięcie poza poprawnym zakresem LOW/HIGH należy traktować jako potencjalną usterkę.

## 7. Diagnostyka

### Wewnętrzna

Monitorowane są:

- element Halla,
- tor analogowy,
- regulator,
- logika cyfrowa,
- pamięć,
- temperatura.

Po wykryciu błędu: VOUT = VPU.

Diagnostyka jest ponawiana co około 2 ms.

### Zewnętrzna

| Usterka | Wynik |
| --- | --- |
| Zwarcie VCC-VOUT | VCC |
| Zwarcie VOUT-GND | GND |
| Zwarcie VCC-GND | VPU |
| Przerwa VCC | VPU |
| Przerwa VOUT | VPU |
| Przerwa GND | VPU |
| Internal Fault | VPU |

## 8. Power-On

Po włączeniu zasilania wyjście pozostaje w stanie VOUT(FAULT) do zakończenia inicjalizacji.

## 9. Maszyna stanów

```text
POWER_ON
   |
   v
NORMAL_LOW <----> NORMAL_HIGH
      ^             |
      |             |
      +----FAULT----+
```

Przejście LOW→HIGH następuje po przekroczeniu BOP. Przejście HIGH→LOW następuje po spadku poniżej BRP. Internal Fault powoduje przejście do FAULT.

## 10. Tabela stanów

| Stan | Warunek | Wyjście |
| --- | --- | --- |
| POWER_ON | Start | VPU |
| NORMAL_LOW | B < BRP | LOW |
| NORMAL_HIGH | B > BOP | HIGH |
| FAULT | Internal Fault | VPU |
| EXTERNAL_FAULT | Niepoprawne napięcie | poza zakresem |

## 11. Zalecenia

- używać wejścia cyfrowego do detekcji stanu,
- okresowo mierzyć VOUT przez ADC,
- traktować FAULT jako błąd krytyczny,
- zgłaszać niepoprawne poziomy napięcia jako usterkę okablowania.
